Pengertian JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi lebih interaktif. JavaScript bekerja di sisi klien (client-side) dan memungkinkan pengguna untuk berinteraksi dengan halaman web tanpa perlu memuat ulang halaman tersebut.

Dengan JavaScript, kita dapat membuat efek dinamis seperti animasi, pengisian otomatis form, serta membuat aplikasi web yang lebih kompleks. JavaScript bekerja bersamaan dengan HTML untuk struktur halaman dan CSS untuk tampilan.

Contoh Program JavaScript

            
                <!DOCTYPE html>
                <html lang="id">
                <head>
                    <meta charset="UTF-8">
                    <title>Contoh JavaScript</title>
                </head>
                <body>
                    <h1>Selamat Datang di JavaScript</h1>
                    <p>Klik tombol di bawah untuk menampilkan pesan:</p>
                    <button onclick="tampilkanPesan()">Klik Saya</button>
                    <script>
                        function tampilkanPesan() {
                            alert('Hello, Selamat datang di JavaScript!');
                        }
                    </script>
                </body>
                </html>

Penjelasan Program:

  • HTML: Menyediakan elemen halaman seperti tombol dengan atribut `onclick` untuk memicu JavaScript.
  • JavaScript: Fungsi `tampilkanPesan()` menampilkan pesan menggunakan `alert()`.

Program di atas membuat tombol yang akan menampilkan pesan pop-up ketika diklik. Ini adalah contoh bagaimana JavaScript dapat membuat halaman web lebih interaktif.
