Pengertian Software

Software adalah sekumpulan program yang mengontrol perangkat keras dan memungkinkan pengguna menjalankan berbagai tugas. Software tidak berwujud fisik tetapi bisa diakses melalui komputer.

Contoh Software

  • Software Sistem: Sistem operasi (Windows, Linux, macOS) dan utility program.
  • Software Aplikasi: Microsoft Office, browser, aplikasi desain.
  • Software Pemrograman: Python, Java, Visual Studio Code.
