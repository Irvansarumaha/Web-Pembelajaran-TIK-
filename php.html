Pengertian PHP

PHP (Hypertext Preprocessor) adalah bahasa pemrograman yang digunakan untuk membuat halaman web dinamis. PHP berjalan di sisi server (server-side) dan dapat menghasilkan konten yang berbeda berdasarkan input dari pengguna atau data yang disimpan dalam database.

PHP sering digunakan untuk mengembangkan situs web interaktif dan aplikasi berbasis web. Dengan PHP, pengembang dapat memproses data form, mengelola sesi pengguna, berinteraksi dengan database, dan masih banyak lagi.

Contoh Program PHP

            
                <?php
                    echo "Selamat datang di PHP!";
                    $nama = "Matius";
                    echo "<br>Halo, " . $nama . "!";
                ?>

Penjelasan Program:

  • <?php ... ?>: Digunakan untuk menulis kode PHP dalam halaman HTML.
  • echo "Selamat datang di PHP!"; Menampilkan pesan ke layar.
  • $nama = "Matius"; Mendeklarasikan variabel `$nama` dan memberikan nilai.
  • echo "<br>Halo, " . $nama . "!"; Menampilkan pesan dengan menyertakan nilai variabel `$nama`.

Program di atas akan menampilkan pesan "Selamat datang di PHP!" dan kemudian menyapa pengguna dengan nama yang disimpan dalam variabel `$nama`.
