Pengertian SQL

SQL (Structured Query Language) adalah bahasa yang digunakan untuk mengelola dan mengakses data dalam sistem manajemen basis data relasional (RDBMS). SQL memungkinkan pengguna untuk melakukan operasi seperti memasukkan data, memperbarui data, menghapus data, dan mengambil data dari basis data.

SQL merupakan standar industri yang digunakan oleh banyak sistem manajemen basis data seperti MySQL, PostgreSQL, SQL Server, dan Oracle. Dengan SQL, pengembang dapat melakukan operasi basis data dengan cara yang efisien dan terstruktur.

Contoh Query SQL

            
-- Membuat tabel
CREATE TABLE Mahasiswa (
    id INT PRIMARY KEY,
    nama VARCHAR(100),
    umur INT
);

-- Menambahkan data
INSERT INTO Mahasiswa (id, nama, umur) VALUES (1, 'Matius', 19);

-- Mengambil data
SELECT * FROM Mahasiswa;

Penjelasan Query:

  • CREATE TABLE Mahasiswa: Membuat tabel baru dengan nama Mahasiswa.
  • INSERT INTO Mahasiswa: Menambahkan data mahasiswa ke dalam tabel Mahasiswa.
  • SELECT * FROM Mahasiswa: Mengambil semua data yang ada dalam tabel Mahasiswa.

Contoh di atas menunjukkan bagaimana membuat tabel, memasukkan data, dan mengambil data dari sebuah tabel menggunakan SQL.
