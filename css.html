Pengertian CSS

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain dan mengatur tampilan atau layout halaman web yang dibangun dengan HTML. CSS memungkinkan kita untuk mengontrol warna, font, spasi, ukuran elemen, dan berbagai aspek visual lainnya dalam halaman web.

CSS dapat diterapkan secara internal dalam HTML menggunakan tag <style>, atau dapat dipisahkan dalam file eksternal dengan ekstensi .css. CSS membantu pengembang web memisahkan struktur (HTML) dan tampilan (CSS) untuk membuat kode lebih bersih dan terorganisir.

Contoh Program CSS

            
                <!DOCTYPE html>
                <html lang="id">
                <head>
                    <meta charset="UTF-8">
                    <title>Contoh CSS</title>
                    <style>
                        body {
                            font-family: Arial, sans-serif;
                            background-color: #f0f0f0;
                        }
                        h1 {
                            color: #ff5722;
                        }
                        p {
                            color: #333;
                            font-size: 18px;
                        }
                    </style>
                </head>
                <body>
                    <h1>Selamat Datang di Halaman CSS!</h1>
                    <p>Ini adalah halaman yang menggunakan CSS untuk penataan.</p>
                </body>
                </html>

Penjelasan Program:

  • <style>: Digunakan untuk menulis kode CSS secara internal dalam halaman HTML.
  • body { font-family: Arial, sans-serif; }: Menentukan font yang digunakan untuk seluruh halaman.
  • h1 { color: #ff5722; }: Mengubah warna teks judul menjadi oranye.
  • p { color: #333; font-size: 18px; }: Mengatur warna teks paragraf dan ukuran font-nya.

Program di atas menampilkan halaman dengan teks yang sudah diberi gaya menggunakan CSS. Bagian h1 akan berwarna oranye, dan teks paragraf akan berwarna abu-abu gelap dengan ukuran font 18px.
