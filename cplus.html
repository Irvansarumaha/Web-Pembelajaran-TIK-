Pengertian C++

C++ adalah bahasa pemrograman yang dikembangkan oleh Bjarne Stroustrup pada tahun 1979 sebagai pengembangan dari bahasa C. C++ mendukung paradigma pemrograman berorientasi objek (OOP) serta pemrograman prosedural. C++ banyak digunakan untuk pengembangan perangkat lunak yang membutuhkan performa tinggi seperti aplikasi desktop, game, serta sistem operasi.

C++ memberikan kontrol yang lebih besar terhadap sistem dibandingkan dengan bahasa pemrograman tingkat tinggi lainnya, dan juga memungkinkan pengembang untuk mengelola memori secara langsung, yang merupakan kelebihan utama dari bahasa ini.

Contoh Program C++

            
#include 
using namespace std;

int main() {
    // Menampilkan pesan
    cout << "Hello Matius!" << endl;
    return 0;
}

Penjelasan Program:

  • #include <iostream>: Menyertakan library untuk input dan output.
  • using namespace std; Menggunakan namespace standar untuk memudahkan pemrograman.
  • cout << "Hello Matius!" << endl; Menampilkan pesan "Hello Matius!" ke layar.
  • return 0; Menandakan bahwa program selesai dijalankan tanpa error.

Program di atas merupakan contoh dasar dalam C++ yang hanya menampilkan pesan "Hello Matius!" di layar.
