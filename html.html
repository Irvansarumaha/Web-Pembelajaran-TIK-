Pengertian HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat dan menyusun halaman web. HTML memberikan struktur dasar untuk konten di situs web, seperti teks, gambar, tabel, dan elemen-elemen lainnya. HTML menggunakan tag-tag untuk menandai bagian-bagian dari konten yang ingin ditampilkan di web. Setiap tag HTML memiliki fungsi tertentu, seperti menampilkan teks, menambahkan gambar, atau membuat tautan antar halaman.

HTML adalah dasar dari pengembangan web dan bekerja bersama CSS (Cascading Style Sheets) dan JavaScript untuk mempercantik dan menambah interaktivitas pada halaman web.

Contoh Program HTML

            
                <!DOCTYPE html>
                <html lang="id">
                <head>
                    <meta charset="UTF-8">
                    <title>Contoh HTML</title>
                </head>
                <body>
                    <h1>Selamat Datang!</h1>
                    <p>Ini adalah halaman web pertama saya.</p>
                </body>
                </html>

Penjelasan:

  • <h1>: Judul utama halaman.
  • <p>: Paragraf teks yang menjelaskan konten halaman.

Program di atas akan menampilkan halaman dengan judul dan deskripsi singkat di bawahnya, yaitu "Ini adalah halaman web pertama saya."
